# RecyclerViewCommons
Common classes for RecyclerView

## Set-up

### Download
grab via Gradle:
```groovy
repositories {
    maven {
        url 'https://github.com/uPhyca/RecyclerViewCommons/raw/master/maven-repo'
    }
}

dependencies {
    compile 'com.android.support:recyclerview-v7:23.1.1'
    compile 'com.uphyca:recyclerviewcommons:0.1.0'
}
```
